# Future_DS_03
# Marketing Funnel & Conversion Performance Analysis

## Project Overview
This project analyzes the Bank Marketing Campaign dataset to understand customer conversion behavior, identify funnel drop-offs, and evaluate channel performance.

## Tools Used
- Google Colab
- Power BI
  

## Dataset
Bank Marketing Campaign Dataset from UCI Machine Learning Repository

## Objectives
- Analyze conversion performance
- Identify drop-off points in the funnel
- Compare performance across channels and time periods
- Provide recommendations to improve conversions

## Key Findings
- There is a major drop-off between contacts and conversions
- Channel performance varies
- Conversion rate differs across groups and months

## Deliverables
- Power BI dashboard
- Analysis notebook
- Final report

## Recommendations
- Focus on high-performing channels
- Improve targeting
- Optimize campaign timing
- Reduce drop-offs in the funnel
